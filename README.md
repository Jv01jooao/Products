## 📥 Como baixar o aplicativo (.exe)

Siga os passos abaixo para baixar o executável diretamente pelo GitHub:

1. Acesse a aba **Releases** deste repositório.
2. <img width="1918" height="767" alt="image" src="https://github.com/user-attachments/assets/6e79ee5d-77b3-455b-850f-e4d7096f9558" />
3. Clique em **UsingViewModelsinMVVM.exe**.
4. Aguarde o download ser concluído.
5. Após baixar, abra o arquivo clicando duas vezes no `.exe`.

⚠️ Caso o Windows exiba um aviso de segurança:

* Clique em **Mais informações**
* Em seguida, clique em **Executar assim mesmo**

✅ O aplicativo é auto suficiente (Self-contained), portanto não é necessário instalar o .NET.

Se tiver qualquer problema, abra uma *Issue* no repositório.
